The Federated Matrix Protocol

What is the Matrix chat protocol?

Imagine your cell phone provider is Verizon. Image then not being able to call people on Sprint or ATT. Ridiculous right?

This is exactly the current state of instant chat.

Enter Matrix. Matrix is a federated chat protocol, which means it can be used by any chat provider as a unified standard. Anyone can run a server, meaning you own your data, not Facebook. Federation means that Matrix servers also can talk to each other meaning inter-community communication, interaction lacking from most of today's social silo'ed echo chamber ecosystems.

Imagine sending messages to Slack via Facebook Messenger or Slack messages to a Discord server. Sync a WeChat room and a Slack room. Send chat commands to your home assistant or home automation. The way chat was meant to be.
